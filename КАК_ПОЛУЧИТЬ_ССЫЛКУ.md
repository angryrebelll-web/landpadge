# Как получить ссылку на сайт Propellini

## Варианты деплоя:

### 1. **Netlify Drop** (Самый простой способ)
1. Перейди на https://app.netlify.com/drop
2. Перетащи папку `deploy` на страницу
3. Получишь ссылку типа: `https://random-name-123.netlify.app`

### 2. **Vercel** (Тоже очень просто)
1. Перейди на https://vercel.com
2. Создай аккаунт
3. Нажми "Add New" → "Project"
4. Загрузи папку `deploy`
5. Получишь ссылку автоматически

### 3. **GitHub Pages** (Если есть GitHub)
1. Создай репозиторий на GitHub
2. Загрузи папку `deploy` в репозиторий
3. В настройках репозитория включи GitHub Pages
4. Выбери папку `deploy` как source
5. Ссылка будет: `https://username.github.io/repo-name/`

### 4. **Cloudflare Pages** (Бесплатно)
1. Перейди на https://pages.cloudflare.com
2. Зарегистрируйся или войди
3. Нажми "Create a project" → "Upload assets"
4. Загрузи папку `deploy`
5. Получишь ссылку

## Структура файлов для деплоя:
```
deploy/
├── index.html      ← Главная страница
├── script.js       ← Логика калькулятора
├── style.css       ← Стили
└── fonts/          ← Шрифты (если нужны)
```

## Важно!
Все файлы (index.html, script.js, style.css) должны быть в одной папке при деплое.

## Быстрый деплой через Netlify CLI:
```bash
npm install -g netlify-cli
cd deploy
netlify deploy --prod
```

## Или создай ZIP файл:
Создай ZIP архив папки `deploy` и загрузи на любой из сервисов выше.
