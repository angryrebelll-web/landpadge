# Рекомендации по поиску изображений для фоновых блоков

## Блоки, требующие изображений

### 1. УТП/Статистика (Stats)
**Тематика**: Автомобили, профессионализм, статистика, достижения
**Поисковые запросы**:
- "luxury car professional service"
- "car detailing statistics"
- "premium car protection"
- "автомобиль премиум класс"
- "профессиональная оклейка автомобиля"

### 2. Акция (Promo) 
**Тематика**: Автомобиль с защитной пленкой, броня лобового стекла
**Поисковые запросы**:
- "car windshield protection film"
- "PPF car wrap"
- "автомобиль с защитной пленкой"
- "броня лобового стекла"

### 3. Калькулятор (Calculator)
**Тематика**: Автомобиль, процесс оклейки, калькулятор стоимости
**Поисковые запросы**:
- "car wrap calculator"
- "PPF installation process"
- "автомобиль оклейка пленкой"
- "процесс оклейки защитной пленкой"

### 4. Виды пленок (Films)
**Тематика**: Различные виды защитных пленок, материалы
**Поисковые запросы**:
- "car protection film types"
- "PPF vinyl wrap materials"
- "виды защитных пленок"
- "антигравийная пленка материалы"

### 5. Отзывы (Reviews)
**Тематика**: Довольные клиенты, качественная работа, автомобили
**Поисковые запросы**:
- "satisfied customer car service"
- "happy car owner"
- "довольный клиент автомобиль"
- "качественная оклейка автомобиля"

### 6. Контакты (Contacts)
**Тематика**: Офис, салон, профессиональная обстановка
**Поисковые запросы**:
- "car detailing shop interior"
- "professional car service office"
- "салон оклейки автомобилей"
- "профессиональный автосервис"

## Использование MCP Google Images Search

Если у вас настроен MCP сервер для поиска изображений, используйте следующие команды:

1. Найти изображения для блока Статистика:
   ```
   Найди 5 изображений: luxury car professional service statistics
   ```

2. Найти изображения для блока Акция:
   ```
   Найди 5 изображений: car windshield protection film PPF
   ```

3. И т.д. для каждого блока

## Альтернативные источники

- **Unsplash**: https://unsplash.com (бесплатные высококачественные фото)
- **Pexels**: https://pexels.com (бесплатные фото)
- **Pixabay**: https://pixabay.com (бесплатные фото)
- **Google Images**: с фильтром "Использование" → "Можно использовать и изменять"

## Требования к изображениям

- **Разрешение Desktop**: 2560x1400px (горизонтальное)
- **Разрешение Mobile**: 1170x2532px (вертикальное)
- **Формат**: WebP с качеством 90%
- **Стиль**: Темные, профессиональные, подходящие для overlay градиента
- **Лицензия**: Убедитесь, что изображения можно использовать коммерчески
